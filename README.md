# Project Setup

## 1. Create a .env File

a. In the root directory of the project, create a file named .env.
b. Inside this file, add the following line:
`BOT_TOKEN=YOUR_BOT_TOKEN`

## 2. Install Dependencies

`pip install -r requirements.txt`

## 3. Run the Bot

`python main.py`
